    🦄 FreeSql

    是一款功能强大的对象关系映射（O/RM）组件，支持 .NET Core 2.1+、.NET Framework 4.0+ 以及 Xamarin✨

    心动不如行动 💡服务支持 💻

    开发者优先

    💡以开发者为中心的设计理念，想你所想，亦享你所享。

    多场景实现

    🛠 支持 CodeFirst / DbFirst / DbContext / Repository / UnitOfWork / AOP / 支持 .NETCore 2.1+, .NETFramework 4.0+, Xamarin

    多数据库支持

    🌳MySql、SqlServer、PostgreSQL、Oracle、Sqlite、Firebird、达梦、人大金仓、神舟通用、南大通用、翰高、ClickHouse、Access 等数据库

    丰富的表达式函数

    ✒ 支持 丰富的表达式函数，以及灵活的自定义解析；

    DbFirst

    💻 支持 DbFirst 模式，支持从数据库导入实体类，或使用实体类生成工具生成实体类；

    类型映射

    ⛳ 支持 深入的类型映射，比如 PgSql 的数组类型等；

    导航属性

    🏁 支持 导航属性一对多、多对多贪婪加载，以及延时加载；

    读写分离

    📃 支持 读写分离、分表分库、过滤器、乐观锁、悲观锁；
